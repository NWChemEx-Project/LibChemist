Detailed Description

template<>
class libchemist::sparse_map::SparseMap< ElementIndex, ElementIndex >

Specializes SparseMap to the case where the independent and dependent indices are both for elements.

This class is a thin addition to SparseMapBase, which adds some useful conversions from other SparseMap types.

Creates a new instance by mapping the dependent tile indices in another instance to their respective element indices.

This ctor will create a new element-to-element sparse map by mapping the tile-index domains in another instance to element-index domains using the TiledRange associated with the Domains.

Parameters
[in]otherThe SparseMap we are converting to a element-to-element SparseMap.
Exceptions
std::bad_allocif there is insufficient memory to create the new instance. Strong throw guarantee.

explicit

Creates a new instance by mapping the independent tile indices in another instance to their respective element indices.

This ctor will map the independent tile indices of another SparseMap instance to there respective element indices. The domain associated with each tile index in other will be associated with each element index in that tile.

Parameters
[in]otherThe SparseMap we are mapping from.
Exceptions
std::runtime_errorif other's TiledRange is not set. Strong throw guarantee.
std::bad_allocif there is not enough memory to allocate the new instance. Strong throw guarantee.

libchemist::sparse_map::SparseMap< ElementIndex, ElementIndex >::SparseMap ( const SparseMap< TileIndex, TileIndex > &  other)
explicit

Creates a new SparseMap by converting the independent and dependent tile indices in another instance.

This ctor creates a new SparseMap from a SparseMap whose independent and dependent indices are both tile indices. The conversion is done by using the TiledRange instances associated with the independent indices and the Domains in other.

Parameters
[in]otherThe SparseMap we are converting.
Exceptions
std::runtime_errorif other contains independent indices of rank 1 or greater, but does not contain a TiledRange. Strong throw guarantee.
std::bad_allocif there is insufficient memory to create the new instance. Strong throw guarantee.
