libchemist::detail_::ChemicalSystemPIMPL::ChemicalSystemPIMPL ( ChemicalSystemPIMPL &&  )
delete

Deleted move ctor.

The move ctor is deleted to avoid slicing the PIMPL's class hierarchy. Moving of the actual ChemicalSystem class is allowed and works by moving from the underlying unique_ptr, which has no chance of slicing.
