Detailed Description

template<typename Geminal>
class libchemist::operators::CorrelationFactor< Geminal >

Models the correlation factor in F12 theories.

In F12 the correlation factor is the operator which couples the electrons together. It is typically either a single Slater-type geminal or a linear- combination of Gaussian functions. The parameters for the correlation factor are stored in the object of type Geminal, whereas the CorrelationFactor class itself models the overall operator.

Makes a new CorrelationFactor equal to the product of this one and rhs.

This function multiples this instance's geminal by the geminal in rhs. The resulting geminal is then returned in a new CorrelationFactor instance.
