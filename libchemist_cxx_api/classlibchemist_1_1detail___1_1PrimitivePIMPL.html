Detailed Description

template<typename T>
class libchemist::detail_::PrimitivePIMPL< T >

Implements the Primitive and PrimitiveView classes.

The PrimitivePIMPL class is responsible for implementing the Primitive and PrimitiveView classes. Within the PrimitivePIMPL class the dispatching between owning and aliasing is handled by the OwnOrBorrow class.

Template Parameters
TThe type used to store the primitive's parameters. Should be non-cv qualified and is expected to be a POD floating point type.

Takes ownership of another PrimitivePIMPL's state.

Assuming T is a POD type, then this ctor will actually deep copy rhs's state if rhs owns its state or shallow copy rhs's state if rhs aliases. In turn the resulting instance will either be a deep copy of rhs or will alias the same Primitive as rhs.

Parameters
[in,out]rhsThe instance to take the state of. After this operation rhs is in the same state as before the ctor.

This ctor is used to create a new PrimitivePIMPL with the specified state. If the inputs are by value or reference the resulting instance will contain a deep copy of the parameters. If the inputs are pointers, then the resulting instance will alias the provided state.
