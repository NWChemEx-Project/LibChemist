Detailed Description

template<typename ReferenceType, typename TensorType>
class libchemist::wavefunction::ManyBody< ReferenceType, TensorType >

Wavefunction for many-body theories.

This wavefunction is designed for use with Moller-Plesset perturbation theory and coupled cluster theory. At the moment it is only capable of holding single and double excitation amplitudes, thus modifications are required for higher-orders.

Template Parameters
ReferenceTypeType of the reference wavefunction.
TensorTypeType of the tensor holding the amplitudes
