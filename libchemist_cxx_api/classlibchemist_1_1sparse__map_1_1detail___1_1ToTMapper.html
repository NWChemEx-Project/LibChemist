[in]ind_tot_mode_2_t_modeA map such ind_tot_mode_2_t_mode[i] is the mode of the tensor that the i-th independent mode of the ToT maps to. Note that a given independent ToT mode may not map onto any mode of the tensor (i.e. it is a free independent mode). ind_tot_mode_2_t is assumed to be a bijection (i.e., ind_tot_mode_2_t_mode[i] does not equal ind_tot_mode_2_t_mode[j] unless i == j).
Exceptions
std::runtime_errorif ind_tot_mode_2_t_mode is not a bijection. Strong throw guarantee.
