Detailed Description

template<typename OverlapType>
class libchemist::orbital_space::DependentBaseSpace_< OverlapType >

Base class for orbital spaces that are dependent on another space.

Particularly in linear scaling theories we often get orbital spaces that depend on another space. That is to say we get a different set of orbitals for each orbital (or tuple of orbitals) in the independent space.

OverlapTypeThe type used to store the overlap matrix.
