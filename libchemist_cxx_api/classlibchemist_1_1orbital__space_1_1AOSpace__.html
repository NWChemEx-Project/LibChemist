Detailed Description

template<typename BasisType, typename BaseType>
class libchemist::orbital_space::AOSpace_< BasisType, BaseType >

Models a space spanned by atomic orbitals.

The AO space takes on a special role in quantum chemistry in that it is a sort of lowest common denominator among spaces. This is because we usually know how to compute integrals in the AO space, but not directly in the derived spaces.

In addition to an overlap matrix the AO space also contains an instance of the parameters which define the atomic orbitals.
