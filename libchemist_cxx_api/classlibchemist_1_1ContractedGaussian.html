Detailed Description

template<typename T>
class libchemist::ContractedGaussian< T >

An atomic orbital resulting from a linear combination of Gaussian primitives.

A contracted Gaussian type orbital formed from primitive gaussians, , is given by:

{ ({r};{c}, {}) = {i=1}^N g({r};c_i, )

  • ({r};{c}, {}) = {i=1}^N c_ie^{- r^2}

}. This class models the resulting orbital by value, meaning instances of this class own the memory associated with their parameters.

TThe non-cv qualified type of the primitive's parameters.

Transfers another ContractedGaussian's state into a new instance.
