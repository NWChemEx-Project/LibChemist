Detailed Description

template<typename... Fields>
class libchemist::detail_::AoSElement< Fields >

The class that wraps an element within the AoSFacade.

User declared classes can not be inserted directly into an array with the guarantee that the members will be contiguous. This is because the class is opaque. The AoSElement is meant to hold the state of the class (the actual class's accessors then being a thin wrapper over the AoSElement) in a manner allowing for the fields to be stored contiguously.

Template Parameters
FieldsThe types of the fields associated with this element.

Adds a value to a particular field.

Since the data is ultimately stored contiguously the values for each element must be together. The easiest way to ensure that is if we assume that the current element is the last element and we are thus adding to the end of the array. This function assumes this and will throw if that is not the case.

Template Parameters

Parameters

Exceptions

std::bad_allocif there is insufficient memory to add the additional value to the buffer. Strong throw guarantee.
