Detailed Description

Describes a distance.

This class is basically just a strong-type. Conceivably it could be fleshed out a bit more to include units or something.

Determines if two Distance instances are different.

Two Distance instances are equal if the values they wrap compare equal. This comparison is done at a binary level thus all bits must be the same, i.e. Distances of 1.230000 and 1.230001 will not compare equal.
