Detailed Description

template<typename VariantType>
class libchemist::tensor::detail_::OpWrapper< VariantType >

Wraps the result of a tensor operation.

This is the common type for the wrapper which results from a tensor expression. It supports further expression chaining to create even more complicated expressions.

Template Parameters
VariantTypeThe type of the std::variant wrapped in this class. The types in the std::variant will be labeled tensor instances, or the types resulting from combining labeled tensors with operations.
