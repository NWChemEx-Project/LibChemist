Detailed Description

template<typename SetType>
struct libchemist::set_theory::FamilyOfSetsTraits< SetType >

Given a set type determines the types for a FamilyOfSets which wraps the set.

Given a set type SetType this class will contain a series of types and static factory functions which the FamilyOfSets will use to implement itself. By specializing this class it should be possible to get FamilyOfSets to work with almost any container. The primary template should work for most containers. All specializations must minimally have the same public API.

Template Parameters
SetTypeThe type of the container the FamilyOfSets will wrap.

template<typename SetType >
using libchemist::set_theory::FamilyOfSetsTraits< SetType >::il_type = std::initializer_list<subset_il_type>

Type of an initializer list which uses offsets to elements.

You can think of family of sets as a list of lists so the structure of the initializer list should be akin to:

il_type il{{0, 1, 2}, {1, 2, 3}};

This example initializer list declares two sets. The first set contains elements 0, 1, and 2. The second set contains elements 1, 2, and 3.

Turns a parent set instance into a pointer to a parent set.

For most containers turning a parent set into a pointer is straightforward; however, for some types, such as tuples, there are a couple of container-specific steps. This function can be used to hide those steps.
