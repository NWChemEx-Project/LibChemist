Detailed Description

A class for managing the available basis sets.

To a large extent this class is envisioned as nothing more than a glorified map from string names of basis sets to the parameters. How those basis sets are stored and/or accessed is a detail left up to the PIMPL backend.

As it stands now, there's not a whole lot of reason to justify making this a class versus a free function; however, it's reasonable to down the line add the ability to say interact with this class as if it really does have state. For example, one can envision inquiring into whether or not a basis set is defined for an atom, or adding additional basis sets dynamically.
