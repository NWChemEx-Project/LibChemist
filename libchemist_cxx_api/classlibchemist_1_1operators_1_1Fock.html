Detailed Description

A class to store and manipulate Fock operators.

This class provides a set of public APIs for the storage and manipulations of generic Fock operators resulting from molecular Hamiltonians representing some physical state of affairs. Fock instances are comprised of a collection of one Electron Operators which represent either physical interactions of the particles of the system or mean-field potentials arising from the interactions of individual Electrons with the average bevhaviour of the other Electrons.
