Detailed Description

Domain containing ElementIndex instances.

This specialization of Domain is for when a Domain is filled with elemental indices. Relative to DomainBase the only added feature is the ability to create a Domain of elemental indices from a Domain of tile indices.

Converts a Domain of tile indices into a Domain of element indices.
