Detailed Description

template<typename OrbitalEnergyType, typename BaseType>
class libchemist::orbital_space::CanonicalSpace_< OrbitalEnergyType, BaseType >

Models an orbital space in which the Fock matrix is diagonal.

For orbital spaces which diagonalize the Fock matrix we can associate an energy with each orbital (specifically the eigenvalue of the Fock matrix associated with that orbital). This class models an orbital space that diagonalizes the Fock matrix in addition to the properties bestowed on the space by BaseType.
