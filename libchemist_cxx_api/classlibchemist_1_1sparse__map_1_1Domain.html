Detailed Description

A set of tile coordinate indices.

Practically speaking a domain collects the dependent tile indices associated with a particular independent element index. The SparseMap class manages the mapping of independent element indices to Domain instances.

Note
Nothing about the machinery of the Domain class actually assumes that the values in the Domain are coordinate indices of tiles. The Domain class simply assumes it is managing indices.
