Detailed Description

template<typename OverlapType>
class libchemist::orbital_space::BaseSpace_< OverlapType >

Models the most fundamental properties of an orbital space.

The BaseSpace_ class contains the properties all orbital spaces have in common. For the moment this amounts to orthonormality. The orthonormality information is stored by means of the overlap matrix.

Note
This class's name ends in an underscore because we intend for users to declare instances of it with typdefs.
