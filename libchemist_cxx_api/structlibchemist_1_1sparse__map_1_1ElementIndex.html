Detailed Description

Strong type signaling that a tuple of integers should be regarded as the index of an element in the tensor.

ElementIndex instances have no additional features/state over an Index instance. The type is simply used at compile time to signal that an index tuple should be treated as indexing an element of a tensor and not a tile of a tensor.
