Detailed Description

Code factorization for a container filled with operators.

LinearCombination stores a series of operators in a type-erased manner. This is accomplished by storing them as pointers to the OperatorBase instances. As a first pass we are going to make the LinearCombination own the operators it is given. This means it will copy them in unless you move them in.
