◆ max_l()

template<typename T>
size_type libchemist::AOBasisSet< T >::max_l ( ) const

Determines the largest total angular momentum of the shells in the basis set.

Many integral libraries preallocate a scratch buffer given the maximum total angular momentum of any particular shell. This function is a convenience function for determining that value for the current basis set.

Returns
The maximum total angular momentum of the shell with the largest total angular momentum.
Exceptions
std::runtime_errorif this basis set does not contain any shells. Strong throw guarantee.
Note
Since a total angular momentum of 0 is a valid result, we instead opt to throw if this function is called on an empty basis set.
