Detailed Description

template<typename OccupiedSpaceType, typename VirtualSpaceType = OccupiedSpaceType>
class libchemist::wavefunction::DeterminantSpace< OccupiedSpaceType, VirtualSpaceType >

Describes the space of determinant wavefunctions which can be made from molecular orbitals.

This class describes the space of orbital determinants that can be formed from a set of orbitals. This class assumes that the alpha and beta orbitals are restricted to having the same form and that determinants are formed by exciting from an occupied space to a virtual space. Additional partitionings of the orbital space and/or relaxation of the spin-restriction are handled by other classees.

Template Parameters
OccupiedSpaceTypeThe type used to model the occupied space. Assumed to be the type of an instantiation of orbital_space::DerivedSpace, or a class derived from one.
VirtualSpaceTypeThe type used to model the virtual space. Same type assumptions as OccupiedSpaceType. Defaults to OccupiedSpaceType.

libchemist::wavefunction::DeterminantSpace< OccupiedSpaceType, VirtualSpaceType >::operator DeterminantSpace< T, U > ( ) const
inline

Enables implicit upcasting of the orbital spaces in a determinant space.

The orbital spaces in a DeterminantSpace are polymorphic. It sometimes occurs that we have an instance of type DeterminantSpace<X, Y> and we want to pass it as if it was of type DeterminantSpace<T, U> (where X derives from T and Y derives from U). This function makes it so such conversions happen automagically.

Template Parameters
TThe type we want to cast OccupiedSpaceType to.
UThe type we want to cast VirtualSpaceType to.
<anonymous>Ensurse via SFINAE that this conversion only participates in overload resolution when OccupiedSpaceType is implicitly convertible to T, and VirtualSpaceType is implicitly convertible to U.
Returns
The current instance casted to a DeterminantSpace<T, U> instance.
Exceptions
NoneNo throw guarantee.
