Detailed Description

template<typename CGType>
struct chemist::basis_set::ContractedGaussianTraits< CGType >

Propagates types associated with a ContractedGaussian of type CGType.

This struct uses inheritance because the total set of types needed for interacting with a ContractedGaussian is the union of the types needed for the primitives making up the ContractedGaussian plus the types exclusive to the ContractedGaussian. *

Template Parameters
CGTypeexpected to be either an instantiation of ContractedGaussian or ContractedGaussianView.
