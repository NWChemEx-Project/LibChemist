Detailed Description

This file contains a class AoSFacade, which aids in implementing an API that is array of structures (AoS) like when the data is actually stored as a structure of arrays (SoA). To this end all data is stored in shared buffers of which each element and the AoSFacade has a reference to (ensuring the buffers are not deleted from under any of them). Elements then store their offsets in the buffer (as well as their spans) and return their values by basic pointer arithmetic.
