Detailed Description

template<typename PointSetType>
class chemist::PointSetView< PointSetType >

Acts like a reference to a PointSet object.

This class is used when you have existing state which is not actually a PointSet object, but you want to use it like it is a reference to the PointSet object.

Template Parameters
PointSetTypeThe cv-qualified type that *this acts lik an alias of.

This ctor is a shallow copy ctor in the sense that the new PointSetView object aliases the same PointSet as other. It is a deep copy ctor in the sense that the resulting PointSetView can be converted into an alias of another PointSet without affecting the PointSet other aliases.

Creates *this by taking ownership of other's state.

This ctor will transfer the internal state of other to *this. After the operation *this will alias the same PointSet that other did and other will act as if it was an alias of an empty PointSet. Note that after this operation *this does not own the state of the aliased PointSet, rather it owns the state used to alias the PointSet.

Parameters
[in,out]otherThe view to take the state from. After calling this method other will be in a state consistent with aliasing an empty PointSet.
