Detailed Description

template<typename ParticleType>
class chemist::qm_operator::Kinetic< ParticleType >

Describes the kinetic energy of a particle of type ParticleType.

This class is essentially a strong type to signify that a one-particle operator describes the kinetic energy of the particle.

Template Parameters
ParticleTypeThe type of the particle. Assumed to be Electron, Nucleus or a type representing an indeterminate number of such particles.
