Detailed Description

Models a space spanned by cartesian axes (R^N).

R^N is the N-dimensional real space spanned by N orthogonal axes. In practice, N is often 3, in which case the basis functions are usually labeled "x", "y", and "z". However, the user may customize the number of basis functions and their labels.
