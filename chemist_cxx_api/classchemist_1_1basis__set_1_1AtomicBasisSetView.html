Detailed Description

template<typename AtomicBasisSetType>
class chemist::basis_set::AtomicBasisSetView< AtomicBasisSetType >

Behaves like a reference to an AtomicBasisSet.

AtomicBasisSet instances own their state. AtomicBasisSetView instances alias their state, but otherwise have the same API as a AtomicBasisSet instance.

Template Parameters
AtomicBasisSetTypeThe type of AtomicBasisSet this is a view of. AtomicBasisSetType is expected to be a (possibly) cv-qualified instantiation of AtomicBasisSet

Creates a new view by deep copying another instance.

Deep copy in this context refers to the state in the view, not what the view is aliasing. In other words, the view resulting from this ctor will be a view of same AtomicBasisSet as rhs, but *this can be made to alias a different AtomicBasisSet without affecting rhs.

Returns the atomic number associated with *this.

All of the shells in *this are centered at a single point in space. Typically there is also a nucleus at this point too. Often the atomic number of that nucleus dictates what shells are in *this. This method can be used to get and set the atomic number.

Returns the i-th AO on the center.

AOs on the center are numbered by flattening out the shells. In other words, if the first shell has n AOs, the first n AOs are the n AOs of that shell. The n + 1-th AO is then the 0-th AO of the second shell and so on.
