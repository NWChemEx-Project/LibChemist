Detailed Description

template<typename ShellType>
class chemist::basis_set::ShellView< ShellType >

Behaves like a reference to a Shell.

Shell instances own their state. ShellView instances alias their state, but otherwise have the same API as a Shell instance.

Template Parameters
ShellTypeThe type of Shell this is a view of. ShellType is expected to be a (possibly) cv-qualified instantiation of Shell
