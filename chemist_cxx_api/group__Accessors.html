Detailed Description

Allow accessing of the elements contained within the array.

The at/get accessors return the requested value by reference/pointer. The resulting reference/pointer is both readable and writable if the current instance is non-const and only readable if the instance is const. Either way, the result is guaranteed to be contiguous so that the following will always compile and work as expected:

AoSElement<double> ei;
//Fill ei
double* pfield = ei.get<0>();
for(size_type i = 0; i < ei.size<0>(); ++i)
assert(pfield[i] == ei.at<0>(i));
