struct chemist::set_theory::detail_::IsFamilyOfSets< T >

Primary template for determining if type T is a FamilyOfSets.

The primary template is chosen whenever T is not an instantiation of FamilyOfSets. When instantiated the primary template of IsFamilyOfSets contains a static constexpr member value set to false.
