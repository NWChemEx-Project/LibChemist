Detailed Description

Class describing super-/sub-set relationship among sets of Nucleus objects.

See https://nwchemex-project.github.io/Chemist/developer/design/chemistry/fragmented_nuclei.html for design details.

Adds a fragment given iterators to a range of indices.

Fragments in general contain more than one atom, thus creation of a fragment typically requires specifying more than one index. In the usual C++ manner, this method accepts two iterators which can be used to obtain the indices for this fragment.

For example, to add a fragment containing atoms 0, 4, and 5 (indicies refer to offsets in the object returned by this->supersystem()) to *this, the syntax is:

std::vector<FragmentedNuclei::size_type> indices{0, 4, 5};
add_fragment(indices.begin(), indices.end());

Adds a fragment given an initializer list of indices.

This method is used primarily in unit testing, but it allows you to add fragments to *this by providing an initializer list of atom indices.

For example, to add a fragment containing atoms 0, 4, and 5 (indicies refer to offsets in the object returned by this->supersystem()) to *this, the syntax is:

add_fragment({0, 4, 5});

bool chemist::FragmentedNuclei::operator== ( const FragmentedNuclei &  rhs) const
inlinenoexcept

Determines if two FragmentedNuclei instances are equal.

In order for two FragmentedNuclei instances to be considered equal they both must either contain a supersystem or they both must not contain a supersystem. If they both contain a supersystem, then additionally they both must contain the same supersystem as well as containing the same fragments of that supersystem.
