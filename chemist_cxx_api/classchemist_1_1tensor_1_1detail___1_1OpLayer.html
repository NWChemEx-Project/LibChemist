Detailed Description

template<typename DerivedType>
class chemist::tensor::detail_::OpLayer< DerivedType >

Base class for operations in the operation layer.

This class basically exists so that we don't have to do a bunch of SFINAE to only enable fully templated versions of operator+, operator-, etc. for the operations in this component of chemist, i.e., without this class we have:

template<typename LHS, typename RHS>
auto operator+(LHS&& lhs, RHS&& rhs) {}

which requires SFINAE so that it's only enabled for choices of LHS and RHS like MultOp<L, R>. With this class we can do:

template<typename LHS, typename RHS>
auto operator+(const OpLayer<LHS>& lhs, const OpLayer<RHS>& rhs){}
Template Parameters
DerivedTypeThe type of the class we're adding to the operations layer. DerivedType is assumed to derive from OpLayer<DerivedType>.
