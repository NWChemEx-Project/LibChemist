Detailed Description

template<typename... SetType>
struct chemist::set_theory::FamilyOfSetsTraits< std::tuple< SetType... > >

Specializes FamilyOfSets to a tuple of sets.

Having tuples of parent sets is convenient when one wants to associate subsets with each other. For example, using a tuple of an AO basis set and a molecule allows us to have subsets which contain an AO basis set piece and a molecule piece. This template specialization enables FamilyOfSets to work with tuples of containers.

Template Parameters
SetTypeA template parameter pack containg the types of the parent set containers.
