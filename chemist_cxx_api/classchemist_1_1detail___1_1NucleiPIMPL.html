Detailed Description

Implements the Nuclei class.

Nuclei is a set of Nucleus objects. Each Nucleus is an atomic number, mass, and a point charge. This class unpacks the atomic number and mass of each Nucleus into a contiguous array, and then unpacks the point charge part into a Charges.
