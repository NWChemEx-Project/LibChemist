Detailed Description

template<typename... Particles>
class chemist::DecomposableDensity< Particles >

Class describing a decomposable density.

This class ties a density to the derived space it was formed from.

Template Parameters
ParticlesThe types of the particles described by this density. At the moment the types are used purely for strong typing purposes. Expected to be unqualified types.
