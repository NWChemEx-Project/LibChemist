Detailed Description

template<typename Particle1, typename Particle2>
struct chemist::operators::SlaterTypeGeminal< Particle1, Particle2 >

Object describing a geminal, which is modeled by a Slater function.

Geminals are a single object used to describe the correlation of two particles. The SlaterTypeGeminal class models a geminal which uses a function of the form , where is a constant.

Creates a new STG with the provided parameters.

The STG has two parameters: the exponent and the scale factor. The scale factor is usually fixed at -1.0 over the exponent; however we allow the user to specify whatever coefficient they want (N.B. the default coefficient is always -1.0/1.2 even when the exponent is not 1.2).
