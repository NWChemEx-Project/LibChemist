struct chemist::orbital_space::detail_::IsCanonicalSpace< CanonicalSpace< OrbitalEnergy, BaseType > >

Specializes IsCanonicalSpace for when T is a CanonicalSpace.

This partial specialization is selected when T is a CanonicalSpace instantiation. When instantiated, this partial specialization will contain a static constexpr member "value" which is set to true.
