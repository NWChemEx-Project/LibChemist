◆ ShellView()

template<typename T >
chemist::ShellView< T >::ShellView

Creates a ShellView that does not alias any Shell instance.

This ctor can be used to create a placeholder ShellView instance. The resulting instance does not alias any Shell instance and will likely segfault if used.

Exceptions
std::bad_allocif there is insufficient memory to create the PIMPL. Strong throw guarantee.
