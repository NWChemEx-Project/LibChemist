Detailed Description

A class for holding the details of an atomic unit.

Atoms are considered the first tier of input to a computational chemistry program. That is, most types of computational chemistry algorithms need to know where in space the atoms are located and what their chemical identity is. This class also additionally holds the mass of the atom and a string identifier.
