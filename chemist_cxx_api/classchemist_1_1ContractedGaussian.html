Detailed Description

template<typename T>
class chemist::ContractedGaussian< T >

An atomic orbital resulting from a linear combination of Gaussian primitives.

A contracted Gaussian type orbital formed from primitive gaussians, , is given by:

@f{ \chi(\vec{r};\vec{c}, \vec{\alpha}) = \sum_{i=1}^N g(\vec{r};c_i, \alpha_i)

  • \chi(\vec{r};\vec{c}, \vec{\alpha}) = \sum_{i=1}^N c_ie^{-\alpha_i r^2}

@f}. This class models the resulting orbital by value, meaning instances of this class own the memory associated with their parameters.

TThe non-cv qualified type of the primitive's parameters.
