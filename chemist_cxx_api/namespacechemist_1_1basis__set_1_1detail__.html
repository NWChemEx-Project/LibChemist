Detailed Description

Contains implementation details for the basis set component.

Source code contained in the basis_set::detail_ namespace is needed for implementing the basis set component, but should not be considered part of the public API. This means that code in the detail_ namespace is subject to change without notice and that users should not directly call it. If as a user you are trying to call code in the basis_set::detail_ namespace you are probably doing something wrong.
