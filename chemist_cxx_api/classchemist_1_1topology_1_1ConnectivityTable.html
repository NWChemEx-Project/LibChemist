Detailed Description

Stores the a list of bonds in a molecule.

From a graph theory point of view this class is essentially an adjacency matrix, i.e., it knows what atoms in the molecule are connected.

At present the table does NOT contain a reference to the Molecule it describes and thus deals entirely in terms of indices.

◆ bonds()

offset_pair_list chemist::topology::ConnectivityTable::bonds ( ) const

Returns a list of the bonds in the connectivity table.

This function will uses the internal representation of the connectivity table to return a list of bonds. Each bond is represented as a pair of zero-based indices such that the pair {i, j} means that atoms i and j are bonded. Pairs are always such that i < j. The final list is sorted lexicographically.

Returns
An nbonds() long list of the bonds in the table. Bonds are sorted in lexicographical order.
Exceptions
std::runtime_errorif there is no PIMPL. Strong throw guarantee.
std::bad_allocif there is insufficient memory to create the return type. Strong throw guarantee.

◆ set_n_atoms()

void chemist::topology::ConnectivityTable::set_n_atoms ( size_type  natoms)

Sets the number of atoms the connectivity table is for.

This works similar to std::vector reserve. Basically it will make the current instance capable of storing connectivity information for natoms. If natoms is not the current the number of atoms that the table is for, the table will reallocate, and be in a state which has no bonds.

Parameters
