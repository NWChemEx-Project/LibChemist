Detailed Description

template<typename MoleculeType>
class chemist::MoleculeView< MoleculeType >

Allows existing molecular state to be used as if it were a Molecule object.

Template Parameters
MoleculeTypeThe type of molecule that *this will alias. MoleculeType is expected to be either Molecule or const Molecule.

template<typename MoleculeType >
chemist::MoleculeView< MoleculeType >::MoleculeView ( const MoleculeView< MoleculeType > &  other)

Creates a new view of the object aliased by other.

This copy ctor will create new instances of the state needed to alias a Molecule object and then point that state at the same Molecule aliased by other. In other words, the resulting view will alias the same Molecule as other, but pointing the resulting view at a different Molecule object will not affect other.

Parameters
[in]otherThe view whose Molecule *this will alias.
Exceptions
std::bad_allocif there is a problem allocating the PIMPL. Strong throw guarantee.
