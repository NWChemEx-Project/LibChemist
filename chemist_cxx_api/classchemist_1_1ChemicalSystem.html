Detailed Description

Class describing the entire chemical system to be modeled.

The Molecule class models the electrons and nuclei in the system; however, there are often additional pieces of the system beyond the molecule. Typically these additional pieces are fields (e.g. electric fields from point charges), but they could also be non-atomistically modeled solvent
