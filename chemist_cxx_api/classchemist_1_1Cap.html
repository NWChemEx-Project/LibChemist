Detailed Description

Caps a severed bond.

See https://nwchemex-project.github.io/Chemist/developer/design/chemistry/caps.html for design details.

bool chemist::Cap::operator== ( const Cap &  rhs) const
inlinenoexcept

Determines if two cap instances are value equal.

Two non-default Cap instances are value equal if they both are anchored to the same atom, replace the same atom, and are comprised of atoms which are value equal (atoms must also be stored in the same order). Cap instances may also be value equal if they are both default instances.

Parameters
[in]rhsThe instance we are comparing *this to.
Returns
True if *this is value equal to rhs and false otherwise.
Exceptions
NoneNo throw guarantee.
