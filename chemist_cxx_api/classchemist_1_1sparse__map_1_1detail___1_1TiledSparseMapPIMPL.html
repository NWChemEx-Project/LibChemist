Detailed Description

template<typename BaseType>
class chemist::sparse_map::detail_::TiledSparseMapPIMPL< BaseType >

Mix-in for adding a TiledRange to a SparseMapPIMPL.

This class is intended to be used as a mix-in to add a TA::TiledRange member to a SparseMapPIPL. The TA::TiledRange is associated with the independent indices (the Domain<TileIndex> class stores its own instance).

Template Parameters
BaseTypeThe class of the SparseMapPIMPL hierarchy that this mix-in should inherit from.

Wraps the BaseType::dp_assign_ to account for TiledRange.

This function wraps BaseType::dp_assign_ ensuring that:

  • both instances have empty TiledRange instances, or
  • both instance have non-empty TiledRange instances

Any other permutation results in an error.
