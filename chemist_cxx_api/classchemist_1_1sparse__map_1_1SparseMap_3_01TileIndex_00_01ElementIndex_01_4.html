Detailed Description

SparseMap from tile indices to element indices.

This SparseMap specialization implements the class when the independent tile indices map to domains of element indices.

explicit

Creates a new tile-to-element SparseMap by expanding the dependent tile indices in a tile-to-tile SparseMap.

This ctor will create a new SparseMap by mapping the dependent tile index domains in other to dependent element index domains using the TiledRange associated with other's domains.

Parameters
[in]otherThe SparseMap we mapping from.
Exceptions
std::runtime_errorif other does not have a TiledRange associated with its Domains. Strong throw guarantee.
std::bad_allocif there is insufficient memory to make the new instance. Strong throw guarantee.

Creates a new instance from a SparseMap<ElementIndex, ElementIndex>

This ctor uses the provided TiledRange to map the independent element indices in the input SparseMap to their respective tile indices. The current instance is initialized to the SparseMap which maps from tile indices to other's element index domains.

Parameters
[in]trangeThe TiledRange to use to convert the indices in other.
[in]otherThe SparseMap we are converting.
Exceptions
std::bad_allocif there is insufficient memory to create this instance. Strong throw guarantee.
std::runtime_errorif the rank of trange does not match the rank of the independent indices in other. Strong throw guarantee.
