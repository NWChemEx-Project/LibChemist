Detailed Description

template<typename SetType>
struct chemist::set_theory::SetTraits< SetType >

Defines the interface that the Subset and FamilyOfSets classes use to interface with the parent object.

The primary template assumes that the parent object has publically accessible:

  • type size_type defining the type the object uses for offsets and indexing.
  • type value_type defining the type of the elements in the object
  • member size() which can be used to retrieve the number of elements in the object.
  • member operator[](size_type) which can be used to retrieve the an element of the object by offset
  • type value_type is comparable via operator==.

If any of the above is not true then using an object of type SetType with the Subset/FamilyOfSets classes requires specializing SetTraits for type SetType.

Template Parameters
SetTypeThe type of the object being subset-ed. To use the primary template definiton SetType must satisfy the criteria listed in the description. Otherwise use of FamilyOfSets and Subset with SetType requires specializing SetTraits.

Defines the API for getting the i-th element in obj.

This function checks that i is less than size(obj), throwing if it is not, and then defers to SetType 's operator[].

Parameters
[in]objThe object we are retrieving an element from.
[in]iThe offset of the desired element. i should be in the range [0, size(obj)).
Returns
A read-only reference to the i-th element in obj.
Exceptions
std::out_of_rangeif i is not in the range [0,size(obj)). Strong throw guarantee.

Defines the API for getting the offset of an element elem.

This function loops over the elements in obj, using get_elem_i, and compares the elements to elem. If a match is found the index of the match is returned. If no match is found an exception is raised.

Parameters
[in]objThe object we are looking for elem in.
[in]elemThe object we are searching obj for.
Returns
The index of elem in obj.
Exceptions
std::out_of_rangeif elem is not in obj. Strong throw guarantee.
???if size(obj), get_elem_i(obj,n) (n less than size(obj), or the value_type::operator== throws. Same throw guarantee.
