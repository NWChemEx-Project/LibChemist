Detailed Description

template<typename T, typename AliasedType>
class chemist::PointView< T, AliasedType >

Implements reference-like semantics with respect to a Point instance.

This class extends the ViewBase API to include forwarding of the Point API.

Template Parameters
TThe type used to hold the coordinates of the aliased Point. Should be cv-qualified to match the Point instance it is aliasing.
AliasedTypeThe type the derived-most class is a view of. AliasedType is assumed to have been derived from Point.

decltype(auto) chemist::PointView< T, AliasedType >::coord ( size_type  q)
inline

Returns the q-th coordinate of the aliased Point.

This function can be used to retrieve one of the components of the aliased Point. The resulting component is read-/write-able if T is non-const. Since the returned component is an alias, modifying it will also modify the parent Point.

Parameters
[in]qWhich component should we return? Must be in the range [0, 3).
