Detailed Description

template<typename ChemicalSystemType>
class chemist::detail_::ChemicalSystemViewPIMPL< ChemicalSystemType >

Defines the API for implementing ChemicalSystemView.

Template Parameters
ChemicalSystemTypeThe type of the ChemicalSystem *this aliases. Assumed to be either ChemicalSystem or const ChemicalSystem.

As of right now there is only a single PIMPL for the ChemicalSystemView, but if others are needed this class will become a base class that defines the API all PIMPLs must implement.
