Detailed Description

A Domain filled with TileIndex instances.

Relative to DomainBase this class has an additional member, the TiledRange. The additional methods provide convenient means of initializing this extra state.

void chemist::sparse_map::Domain< TileIndex >::set_trange ( const tiled_range_type &  trange)

Allows a user to manually set the TiledRange.

Particularly when using one of the base class's ctors, it is possible to create a new Domain which does not have a TiledRange associated with it. This method allows you to manually set the TiledRange.

Parameters
