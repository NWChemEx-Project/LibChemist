chemist::IntegralFactory::IntegralFactory ( const IntegralFactory &  other)

Makes a new IntegralFactory by deep copying another one.

The copy ctor makes a member-wise deep copy of another IntegralFactory instance. Note that if the other IntegralFactory is in a null state (i.e. has no PIMPL) the resulting instance will also be in a null state.

Parameters
[in]otherThe instance to deep copy.
Exceptions
std::bad_allocif an error arises while allocating the new state. Strong throw guarantee.

◆ IntegralFactory() [3/3]

chemist::IntegralFactory::IntegralFactory ( IntegralFactory &&  other)

Takes ownership of another IntegralFactory's state.

The move ctor will transfer the state of other into the newly created IntegralFactory instance. After the move other will be in a null state (i.e. have no PIMPL). N.B. if other was in a null state to begin with the newly created IntegralFactory instance will also be in a null state.

Parameters
[in,out]otherThe instance whose state is being taken. After the construction other will be in a null state.
Exceptions
noneNo throw guarantee.
