Detailed Description

A class for storing a list of pre-built molecules.

This class is largely intended to serve as a glorified map between molecule names and geometries. Molecules must be added to the manager using the MoleculeManager::insert method, since the loaded molecules are no longer controlled by changing out the PIMPL.

As it currently stands there is little to justify making this a class, but conceptually one can envision interacting with this class as a sort of search engine for geometries, e.g., give me all molecules marked as part of the G2 test set, or molecules that don't contain carbon.
