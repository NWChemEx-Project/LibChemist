Detailed Description

template<typename NucleiType>
class chemist::detail_::NucleiUnion< NucleiType >

Implements a NucleiView that is a union of NucleiView objects.

Template Parameters
NucleiTypeThe cv-qualified Nuclei type that *this will act like it aliases.

This PIMPL is useful when you want to use one or more NucleiView objects as if they were a single NucleiView object, but don't want to rearrange the underlying state.

Warning
At present, the resulting union can contain duplicates. This is consistent with the Nuclei class itself which allows duplicates.

Creates a new view containing copies of the views in other.

This ctor will initialize *this with copies of the NucleiView objects in other. The resulting copies still alias the same Nuclei objects, but causing the NucleiViews in *this to alias different Nuclei objects will NOT affect other.
